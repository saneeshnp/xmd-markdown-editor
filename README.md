# XMD Markdown Editor

X Markdown is probably the most simple browser based markdown editor out there.

It provides:

- A plain text editing view for writing markdown
- A preview mode for reading and editing the rendered output within markdown
- Live document details such as word count, character count, reading time, and line count
- Local storage support so your draft stays available after refresh
- A theme toggle and a clear button

## Demo

Live Demo: [XMD Markdown Editor](https://saneeshnp.github.io/xmd-markdown-editor/)

## Usage

Open `index.html` in a browser to use the editor locally. Or just use the demo URL.

## Best use case

Use it as a markdown clipboard. A place to do any quick ad-hoc edits.
